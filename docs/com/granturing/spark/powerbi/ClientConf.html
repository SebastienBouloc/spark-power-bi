com.granturing.spark.powerbi

ClientConf

case class ClientConf(token_uri: String, resource: String, uri: String, username: String, password: String, clientid: String, timeout: Duration) extends Product with Serializable

Specifies the client configuration options for connecting to the PowerBI service. See com.granturing.spark.powerbi.ClientConf.fromSparkConf for generating from Spark configuration.

token_uri

the OAuth token URI to authenticate against

resource

the OAuth resource to authenticate for

uri

the PowerBI API URI

username

the Azure Active Directory account name to authenticate with

password

the Azure Active Directory account password to authenticate with

clientid

the OAuth client id

timeout

the response timeout in seconds for API calls
